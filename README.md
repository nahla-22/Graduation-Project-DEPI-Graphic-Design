SANA Academy - Visual Identity & Social Campaign

A complete visual identity system for SANA Academy, a modern kids learning platform focused on Coding, Design, and Robotics for ages 6–16.

🎯 Project Overview

SANA Academy is designed to bridge the gap between education and creativity, offering a learning experience that is:

Engaging for kids
Trustworthy for parents
Future-oriented and tech-driven

This project focuses on building a balanced identity that feels fun without being childish, and modern without losing warmth.

💡 Concept & Idea

The core idea behind SANA Academy is:

“Where Bright Minds Begin”

The visual identity reflects:

Growth and curiosity
Movement and energy
A safe, inspiring learning environment

The logo symbol (jumping child + circular motion) represents:

Exploration
Freedom of learning
Continuous development
🎨 Visual Identity System
Color Palette
Deep Blue → Trust, stability
Teal → Technology, growth
Orange → Energy, creativity (used as accent)
Yellow → Highlights and attention points

A controlled color system ensures visual consistency across all applications.

Typography
Clean, modern sans-serif typefaces
Strong hierarchy for readability
Arabic + English compatibility
Designed to be clear for both kids and parents
Logo System
Primary logo (full version)
Clear space and usage rules
Scalable across digital and print

⚠️ The logo maintains consistent proportions and typography across all applications.

🧠 Design System

The project follows a structured design approach:

Grid-based layouts (Swiss-inspired)
Consistent spacing system
Modular components
Balanced use of whitespace

This ensures the identity is:

Scalable
Organized
Easy to apply across different media
📱 Social Media Campaign

A 3×3 Instagram grid campaign was designed to:

Capture attention quickly
Communicate value clearly
Maintain a unified visual story
Key Features:
Seamless grid composition
Center-focused hero visual
Combination of text + engaging visuals
Clear CTA (Call to Action)

The campaign balances:

Visual attraction for kids
Clarity and trust for parents
🧩 Brand Applications

The identity was applied across multiple touchpoints:

Stickers
Packaging
Social media posts
Promotional materials

Each application follows the same visual language and system rules.
